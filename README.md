# Expanding Card UI

This project demonstrates a simple and interactive expanding card layout built with **HTML**, **CSS**, and **vanilla JavaScript**.  
The design focuses on responsive behavior, smooth transitions, and clean code structure, making it a great example of a lightweight UI component without external libraries.

## Features
- Expanding and collapsing cards with smooth animations  
- Responsive design for different screen sizes  
- Built using pure HTML, CSS, and JavaScript (no frameworks)

## Installation
Clone the repository and open `index.html` in your browser:
bash
git clone https://github.com/navidmostafaei/expanding-card-vanilla-js.git
cd expanding-card-vanilla-js
open index.html
